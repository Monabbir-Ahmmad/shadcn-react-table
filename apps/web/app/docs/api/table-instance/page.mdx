# Table instance (`table.tableInstance`)

`useDataTable` returns the standard TanStack table instance with one addition: a **`tableInstance`** object holding the table's resolved configuration and UI state. Components read from it instead of prop drilling; you can read and call into it too.

```tsx
table.tableInstance.beginCreate()       // open the create-row form
table.tableInstance.setDensity("compact")
const isLoading = table.tableInstance.isLoading
```

<ApiTable of="tableInstance" propLabel="Member" />

<Callout variant="note" title="Everything else is TanStack">
  All standard instance methods — `getState`, `getRowModel`, `getColumn`,
  `setPageIndex`, `getFilteredRowModel`, `resetSorting`, … — are available
  directly on `table` as usual.
</Callout>
